# Contributing to TrueDeal

## Our Philosophy
TrueDeal is built on **equity, reciprocity, and transparency**.  
All contributions should align with these core values.

## Contribution Areas
1. **Protocol Development** (Solidity, Rust, Zero-Knowledge proofs)
2. **UI/UX Design** (wallet interfaces, dashboards)
3. **Documentation** (technical specs, user guides)
4. **Community Building** (outreach, education, governance)
5. **Ethical Oversight** (compliance, auditing, philosophy)

## First Steps
1. Read our [Manifesto](docs/MANIFESTO.md) and [Ethics Charter](docs/ETHICS_CHARTER.md)
2. Browse open [Issues](https://github.com/TrueDeal/TrueDeal-Protocol/issues)
3. Join our [Discord/Signal] for discussion

## Development Workflow
1. Fork the repository
2. Create a feature branch
3. Make changes with clear commits
4. Submit a Pull Request with detailed description

## Code of Conduct
Be respectful, transparent, and equity-focused.  
We welcome diverse perspectives but require alignment with TrueDeal's ethical foundation.
